# Privacy-first 911 Voice Simulator

This project was built to help students with developmental disabilities practice emergency calls in a safe environment.

The idea came from my mom, a special education teacher, who needed a way for her students to learn how to respond during natural disasters. Many of them cannot read and cannot safely practice calling 911 in real life.

This tool simulates a 911 call in Spanish and uses voice activity detection (VAD) to move forward when the student stops speaking.

Key goals:
- Accessibility
- Confidence and familiarity
- Privacy-first (no audio is stored)
- Runs locally in the browser

This is an early version and I’m actively improving the voice detection and UX.

Feedback is welcome!
